Getting Started

Installation via CDN
jsDelivr
<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/animflow@latest/dist/css/AnimFlow.css">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/animflow@latest/dist/js/AnimFlow.js"></script>
unpkg
<!-- CSS -->
<link rel="stylesheet" href="https://unpkg.com/animflow@latest/dist/css/AnimFlow.css">

<!-- JavaScript -->
<script src="https://unpkg.com/animflow@latest/dist/js/AnimFlow.js"></script>
Installation via npm
npm install animflow
// Require CSS
const AnimFlow = require('animflow/dist/css/AnimFlow.main.css');

// Require JavaScript
const AnimFlow = require('animflow/dist/js/AnimFlow.main.js');

Basic Usage

Add animations to any element using data attributes:

<div data-anim="fade-in">
    This will fade in when scrolled into view
</div>

<script>
    const animFlow = new AnimFlow({
        duration: 1000,
        once: true
    });
</script>
Live Example: This will fade in

Configuration

Customize AnimFlow behavior with these options:

const animFlow = new AnimFlow({
    duration: 1000,     // Animation duration in ms
    delay: 0,          // Base delay for animations
    once: true,        // Only animate once
    mirror: true,      // Reverse on scroll up
    stagger: 200,      // Delay between group items
    easing: 'cubic-bezier(0.4, 0, 0.2, 1)',
    threshold: [0, 0.2] // Intersection thresholds
});
Option Type Default Description
duration number 1000 Duration of the animation in milliseconds
delay number 0 Base delay before animation starts
once boolean true Whether to run the animation only once
mirror boolean true Whether to reverse the animation when scrolling up
stagger number 200 Delay between animations in a group
